## Principle of Mathematical Induction (PMI) 
##### Steps
- **Base Case:** Verify statement for n = 1. 2. 
- **Inductive Hypothesis:** Assume true for n = k. 3. 
- **Inductive Step:** Prove true for n = k+1. 4. 
- **Conclusion:** Statement holds for all n âˆˆ N.
